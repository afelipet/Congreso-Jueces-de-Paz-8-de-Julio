## 🌐 Publicar en GitHub Pages

1. Sube TODO el contenido de esta carpeta (con sus subcarpetas `docs/` y `assets/`) a un repositorio de GitHub.
2. **Settings → Pages** → Source: rama `main`, carpeta `/ (root)` → **Save**.
3. En 1–2 minutos tu sitio queda en `https://[tu-usuario].github.io/[nombre-repo]/`.

> El mapa de Google Maps se ve embebido y funcional una vez publicado en GitHub Pages (en vista previa local puede aparecer un aviso de red, es normal).

## ✏️ Cómo modificar después

- **Texto / agenda / perfiles:** abre `index.html` en GitHub → lápiz (✏️) → edita → **Commit changes**. Los perfiles de ponentes están en el bloque `const speakers = [...]` del `<script>`.
- **Reemplazar un logo:** sube un PNG con el mismo nombre a `assets/`.
- Tras cualquier cambio recarga con **Ctrl+Shift+R** si no ves la actualización.
